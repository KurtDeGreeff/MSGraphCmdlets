# Microsoft Graph API Cmdlets
## Features
These cmdlets are meant to give administrators an easy way to interact with
 the Microsoft Graph API. The hope is to have the most popular features of
  MS Graph here as cmdlets.

### Current Functions
1. Get-GraphUsers
2. Get-GraphGroups
3. New-GraphUser
